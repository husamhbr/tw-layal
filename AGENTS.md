# Layal Component Bundle Rules

- This repository is one Salla Twilight Component Bundle.
- The bundle contains multiple independent homepage sections.
- One component equals exactly one homepage section/template.
- Do not treat the repository as a full Twilight theme.
- Do not modify headers, footers, product pages, category pages, cart, account pages, or full-theme layouts.
- Preserve the Layal bundle identity.
- Build and validate one component at a time.
- Use shared Layal design tokens where appropriate instead of duplicating visual values across components.
- Support RTL and LTR.
- Design mobile independently, not as a compressed desktop layout.
- Prioritize accessibility, performance, and merchant-friendly settings.
- Do not commit, push, publish, or create branches unless explicitly asked.
- Do not create another Salla bundle, theme, landing page, or repository.
